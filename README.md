## Scope of the Project

The scope of the Toll Booth Vehicle Count System includes:

- Counting different types of vehicles passing through a toll booth
- Validating user input to ensure correct vehicle classification
- Displaying a summary of vehicle data in a clear format
- Helping beginners understand core C programming concepts

### Future Enhancements (Out of Scope for Current Version)
- Toll fee calculation for each vehicle type
- Storing vehicle data in files
- Graphical User Interface (GUI)
- Real-time sensor-based vehicle detection
- Daily/Monthly report generation

The current version is limited to console-based input and output and is intended for educational purposes only.
